# Play Tennis - Beginner ML Project 🎾

An end-to-end Machine Learning system that predicts whether to play tennis based on weather conditions.

## Tech Stack
- **Model:** Scikit-learn (Random Forest)
- **Backend:** FastAPI (Python)
- **Frontend:** HTML, CSS, JavaScript

## How to Run
1. Create a virtual environment: `python -m venv venv`
2. Install requirements: `pip install -r requirements.txt`
3. Run training: `python backend/train.py`
4. Start backend: `uvicorn backend.main:app --reload`
5. Open `index.html` via Live Server.
